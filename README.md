# 💰 Income Prediction Using Machine Learning

## 📌 Project Overview

This project focuses on predicting an individual's **Income** based on various demographic and financial attributes using **Machine Learning techniques**. The goal is to analyze the dataset, perform **Exploratory Data Analysis (EDA)**, preprocess the data, and build a predictive model that can accurately estimate income.

The project demonstrates the complete **Data Science workflow**, including data cleaning, visualization, feature engineering, model building, and evaluation.

# 🤖 Machine Learning Model

The model used for prediction:

### Random Forest Regressor

Random Forest was chosen because:

* It handles non-linear relationships well
* Works effectively with mixed data types
* Reduces overfitting through ensemble learning

---

# 📈 Model Evaluation

The model performance was evaluated using:

* **R² Score**
* **Mean Absolute Error (MAE)**
* **Mean Squared Error (MSE)**

These metrics help determine how accurately the model predicts income.
